# SNS 관리 — 마케팅 뚝딱 메이커 바로가기 작업지시서

**대상:** SNS 관리 화면에서 **마케팅 뚝딱 메이커**를 언제든 바로 열어 쓰게 한다(바로가기 방식).
> 🎯 방식: **새 탭으로 여는 바로가기**(별도 BYOK 앱이라 통합 대신 연결이 안전·깔끔). 입장코드·노션 가이드도 함께 안내.

---

## 1. SNS 관리에 "마케팅 뚝딱 메이커" 카드 추가 ⭐
- SNS 관리 화면 상단에 **카드(또는 버튼)** 하나:
  - 제목: **마케팅 뚝딱 메이커**
  - 한 줄 설명: "학원 블로그 글·카드뉴스를 우리 학원 색·말투로 자동 생성"
  - 버튼 ①: **[열기]** → 마케팅 뚝딱 메이커 **새 탭**으로 열림.
  - 버튼 ②: **[사용 가이드]** → 노션 가이드 **새 탭**으로 열림.
  - **입장코드 표시:** `입장코드: bahkkum` (옆에 '복사' 버튼 있으면 편함).
- 깔끔하게 한 카드 안에 정리(군더더기 없이).

## 3. 동작
- [열기]·[사용 가이드] 모두 **새 탭**(현재 작업 흐름 안 끊기게).
- 누구 권한까지 보이게 할지: 기본 **강사·원장**(SNS 관리 접근자). 학생 제외.
- *완료 기준:* SNS 관리에서 [열기]로 마케팅 뚝딱 메이커가 새 탭에 뜨고, [사용 가이드]로 노션 가이드가 열리며, 입장코드 `bahkkum`이 보인다.
