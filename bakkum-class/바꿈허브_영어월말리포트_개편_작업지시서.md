# 초등영어 월말리포트 개편 + 월말리포트 UI 개선 작업지시서

**대상:** 초등영어 월말리포트를 업로드한 **3장 분할 양식**으로 제작 + 입력 화면 개선. 저장/일괄저장 동선은 **수학 월말리포트와 유사**하게.
> 🔓 **이번엔 수학 UI도 개선 허용:** 수학의 **데이터·계산(인센티브 정산 등)은 절대 불변**, 단 **화면(UI)은 개선 가능**. 영어와 같은 디자인 언어로 통일.

---

## 1. 초등영어 월말리포트 — 출력 양식 (3장 분할) ⭐
> ⚠️ **핵심 문제:** 현재 앱이 만드는 영어 월말리포트 이미지는 **약자 배지(P 등) 카드만** 있어 밋밋하고 한눈에 안 들어온다. **원래 양식의 레이더 차트(Point Spread)가 빠졌다.** → **레이더 그래프를 반드시 복원**해서, 학생의 강·약점이 시각적으로 보이게 할 것. (배지 카드 = 1장 표, 그래프 = 2장. 둘 다 있어야 함)

한 학생당 **3장**이 각각 이미지로 저장(업로드한 원래 양식 그대로). 파일명 예: `성적표_김예담_Ivan_2026년_6월_1/2/3.png`.

### 1장 — 표지 + 항목별 평가표
- **헤더(파란 배경):** 졸업모 아이콘 + `바꿈영수학원` / `Bakkum English & Math Academy`. 우측 카드 `Student Progress Report / 성적표`.
- **학생 정보:** 학생(이름+영어이름, 예 `김예담 Ivan`) · 선생님 · 평가월(`2026년 6월`) · 학년(`Grade 초6`).
- **Categorized Evaluation 표** — 3열: `CRITERIA / SCORE / GRADE`
  - 8개 항목(각 영어명 + [한글]):
    Listening[듣기] · Reading[읽기] · Speaking[회화·발표] · Spelling·Writing[철자·영작] · Comprehension[이해] · Learning Attitude[태도] · Task Performance[수행] · Confidence[자신감]
  - SCORE = 등급 약자 **배지**(P/E/GR/G/VG/NI), GRADE = 풀이름(Perfect/Excellent/…).

### 2장 — 레이더 차트 + 등급 범례
- **헤더:** 로고 + 우측 `김예담 Ivan · 2026년 6월`.
- **Point Spread Analysis:** 8축 레이더(축: Listening·Reading·Speaking·Spelling & Writing·Comprehension·Attitude·Performance·Confidence), 0~6 스케일. 학생 점수 면 채움 + **비교선(점선)**.
- **GRADING SCALE 범례(파란 박스):** `P Perfect · E Excellent · GR Great · G Good · VG Very Good · NI Need Improvement`.
- **푸터:** `Academic Assessment Report · Bakkum Academy` / 우측 날짜(`2026년 6월 14일`).

### 3장 — 선생님 코멘트
- **헤더:** 로고 + `김예담 Ivan · 2026년 6월`.
- 제목 `Teacher's Comments · 선생님 코멘트`.
- **코멘트 본문**(자유 서술, 학부모 안내체) — 카드 안에 넉넉한 여백으로.
- 푸터 동일.

> 데이터 출처: 노션 「초등 리포트 DB」(8개 항목 P~NI + Comments). 등급 약자↔레이더 점수 매핑은 기존 그대로.

## 2. 입력 화면 개선 — "옆으로 늘어짐" 해결 ⭐
- **문제:** 8개 항목 등급 입력이 **가로로 길게 늘어져** 입력 불편(가로 스크롤·시선 이동 많음).
- **개선:**
  - 한 학생 = **세로 카드**. 8개 항목을 **세로 목록**으로 쌓고, 각 항목 오른쪽에 **등급 선택**(P/E/GR/G/VG/NI 버튼 토글 또는 드롭다운). **가로 스크롤 없이** 한눈에.
  - **코멘트**는 항목 아래 **넓은 textarea**(세로로 충분히).
  - 학생 전환: **왼쪽 학생 목록 + 오른쪽 입력 패널**(또는 이전/다음 학생 버튼). 한 명씩 빠르게.
  - 좁은 화면·태블릿에서도 세로로 자연스럽게 흐르게.
- *완료 기준:* 가로 스크롤 없이, 한 학생 8개 항목+코멘트를 위→아래로 빠르게 입력한다.

## 3. 이미지 저장 — 수학 월말리포트와 같은 흐름 ⭐
- **미리보기:** 저장 전 **3장 미리보기**(수학처럼 확인 후 저장).
- **개별 저장:** 학생별로 3장 이미지 저장.
- **일괄 저장:** 월·반 선택 → **전체 학생을 3장씩 한 번에** 저장(학생별 묶음/zip, 파일명 규칙 위와 동일).
- 버튼 위치·동선·레이아웃을 **수학 월말리포트와 유사**하게(일관성).
- 대상은 **초등영어만**.
- *완료 기준:* 수학 월말리포트와 같은 자리·흐름에서, 초등영어 3장 양식이 미리보기→개별/일괄 저장된다.

## 4. 수학 월말리포트·대시보드 UI 정리 (데이터 불변)
- 이제 수학도 **화면 개선 허용** → 영어와 **같은 디자인 언어**(여백·카드·배지·버튼·날짜 컨트롤)로 통일.
- **단, 수학 데이터·계산(인센티브 정산·재적 집계 등)은 변경 금지.** 보이는 것만 다듬기.
- *완료 기준:* 수학·영어 월말리포트가 같은 톤으로 보이되, 수학 수치·로직은 그대로.

## 작업 순서
1. **레이더 차트(그래프) 복원**(1번) — 가장 중요. 지금 빠진 시각 그래프부터.
2. **초등영어 3장 양식 완성**(1번 전체: 표지+평가표 / 레이더+범례 / 코멘트).
3. **미리보기·개별/일괄 저장**(3번, 수학과 동일 흐름).
4. **입력 화면 세로 개선**(2번).
5. **수학 월말리포트 UI 통일**(4번, 데이터 불변).
6. 회귀 확인(수학 **데이터** 변화 없음).
