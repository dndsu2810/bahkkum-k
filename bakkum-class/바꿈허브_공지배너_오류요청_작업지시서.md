# 바꿈 허브 — 공지 배너 + 오류/개선 요청 작업지시서

**대상:** ①원장이 강사에게 띄우는 상단 공지 배너 ②모두가 쓰는 오류·개선 요청 창구(원장 **카카오워크**로 알림).
> 🎯 **핵심 가치:** 편리하고, 누구나 이해되고, **군더더기 없이 깔끔**하게. 안 쓰면 안 보이고, 쓸 땐 명확하게.

> ✅ 참고(현재 상태): 헤더에 알림 종(🔔) 있음 · 상단에 초록 알림 배너 자리 이미 있음(시간표 변경 승인 표시) · 로고/원장 표기/영어 그룹명 통일됨. **공지 배너는 이 배너 자리 패턴을 재활용**하면 자연스럽다.

---

## 1. 공지 배너 (원장 → 강사 공지) ⭐
- **용도:** 원장이 강사들에게 알릴 게 있을 때 상단에 띄우는 띠 배너.
- **표시 규칙:** **공지가 있을 때만** 상단에 뜨고, **없으면 아예 안 보임**(빈 배너·자리 차지 X).
- **작성(원장 전용):** 짧은 공지 문구 입력 → 게시. 필요 시 시작/종료일 또는 수동 내리기.
- **위치:** 모든 화면 상단(현재 초록 배너 자리와 같은 위치). 강사·데스크 로그인 시 보임. (학생 노출 여부는 기본 제외)
- **모양:** 한 줄 띠. 📢 아이콘 + 문구. 닫기(×) 누르면 그 사람 화면에선 접힘(다음 게시 전까지). 색은 공지(파랑)·중요(주황) 정도만.
- *완료 기준:* 원장이 공지를 올리면 강사 상단에 한 줄로 뜨고, 내리면 사라진다. 공지 없을 땐 배너 자체가 없다.

## 2. 오류·개선 요청 페이지 ⭐
- **용도:** **누구나**(강사·데스크·**학생 페이지 포함**) 오류나 개선 아이디어를 빠르게 올리는 창구. 원장이 빨리 고치는 통로.
- **작성 폼(군더더기 없이 핵심만):**
  - **어떤 페이지/화면**인지 — 선택(드롭다운: 오늘·출결·숙제·시간표·학생명단·월말리포트·학생화면 등) 또는 자동(현재 보던 화면 추정).
  - **작성자** — **로그인 계정으로 자동 입력**(직접 안 씀).
  - **무슨 문제 / 어떤 개선이면 좋을지** — 자유 입력(한 칸).
  - (선택) 스크린샷 첨부.
- **원장 카카오워크로 알림:** 요청이 등록되면 **원장 카카오워크로 메시지 전송**(작성자·페이지·내용·시간 포함).
  - *구현 메모:* 카카오워크 **웹훅(Webhook)형 봇**으로 전송(이 용도 전용 봇 1개 생성). 봇은 개수 제한 없음. 봇 Webhook URL은 설정에 저장(노출 금지). 받을 채팅방: 원장 1:1 또는 '오류 신고' 채팅방.
- **상태 표시:** `접수 → 해결 중 → 완료`. 원장이 상태를 바꾸면 목록에 반영(완료 시 '완료' 배지).
  - 원장이 '해결 중'으로 두면 **해결 중**으로 보이고, '완료/개선완료'로 바꾸면 **완료** 표기.
  - (선택) 완료 시 작성자에게 앱 내 알림.
- **목록 화면:** 최신순, 상태 필터(접수/해결 중/완료). 원장은 전체·상태 변경 가능, 작성자는 본인 글·상태 조회.
- **메뉴 위치:** 공통(모두 접근). 학생 화면에도 "오류 신고" 진입 한 곳.
- *완료 기준:* 누구나 페이지 선택 + 내용 작성 → 등록하면 원장 카카오워크에 알림이 오고, 원장이 상태(해결 중/완료)를 바꾸면 목록에 그대로 보인다.

---

## 작업 순서
1. **오류·개선 요청 페이지 + 카카오워크 알림 + 상태(접수/해결중/완료)** — 빠른 대응 창구라 우선.
2. **공지 배너**(있을 때만, 원장 작성).
3. 회귀 확인(수학 데이터 불변).

## 체크리스트
- [ ] 공지 배너: 공지 있을 때만 상단 표시, 없으면 미표시 / 원장 작성·내리기
- [ ] 오류·개선 요청: 페이지 선택 + 작성자 자동 + 내용 입력(누구나, 학생 화면 포함)
- [ ] 등록 시 **원장 카카오워크로 알림**(작성자·페이지·내용·시간)
- [ ] 상태 `접수 → 해결 중 → 완료` 표시·원장 변경
- [ ] 목록 최신순·상태 필터, 깔끔한 빈 상태
- [ ] 수학 데이터 회귀 없음

## 확인 필요 (한 가지)
- 알림을 받을 **카카오워크 봇 Webhook URL** — 지현님이 '오류 신고용' 웹훅 봇 1개를 만들어(원장 1:1 또는 '오류 신고' 채팅방에 연결) 그 URL을 설정에 넣는 방식으로 진행(만드는 법은 따로 안내 가능).
