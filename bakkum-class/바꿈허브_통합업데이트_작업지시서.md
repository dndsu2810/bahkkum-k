# 바꿈 허브 — 통합 업데이트 작업지시서

**검토:** 정민아·원장 계정으로 전 화면 재점검(2026-06-14). 아래는 「디테일(문구·디자인)」·「UX 피드백」 문서를 **하나로 합치고**, 새로 발견한 것(홈·원장 표기·시간표 변경 요청)을 더한 최신본.

> ⚠️ **절대 원칙 — 수학 불변:** 수학 화면·데이터·동작은 바꾸지 않는다. 통일은 영어를 수학 패턴에 맞추는 방향.
> 🎯 **목표:** "더 불편하거나 어렵지 않게." 깔끔하고, 한눈에 알고, **쓸 수밖에 없게** 편하게.

## ✅ 이미 반영됨(확인)
사이드바 카테고리 분리 · 사이드바 상단 "정민아님 · 원장"(담당 과목 빠짐) · 전역 학생 검색 · 학생 명단(이름 변경) · 시간표 변경 요청(폼) · 원장 대시보드 · 영어 대시보드 '현황'(인센티브 없음) · 강사 업무 보드 카드 상세/담당/마감 · 학원 일정 공용+노션 가져오기 · 즐겨찾기(★).

## 1. 홈 화면 재정리 — "뭐하는 곳인지" 한눈에 ⭐
- **문제:** 부제가 `담당: 수학 · 원장 · 왼쪽 메뉴에서 화면을 선택하세요.`로 뒤섞여 혼란. 카드 설명도 추상적이라 홈이 비고 어수선.
- **수정(원장 기준):**
  - 인사: `안녕하세요, 정민아 님` (담당 과목 표기 빼기 → 2번)
  - 한 줄 목적: `바꿈영수학원을 한 곳에서 관리하는 공간이에요.`
  - 그 아래 **바로가기 타일 4개**: `오늘` · `학생 명단` · `강사 업무 보드` · `원장 대시보드` (자주 쓰는 곳으로 직행)
  - (선택) 작은 오늘 요약: 오늘 등원/할 일 개수 한 줄.
- 길게 설명하지 말고 **핵심만**. 역할별로 타일은 달라도 형태는 동일.
- *완료 기준:* 홈만 봐도 "여기서 뭘 하는지 + 어디로 가면 되는지"가 3초 안에 이해된다.

## 2. 원장 '담당 과목' 표기 제거 (전역) ⭐
- 원장 계정에는 `담당: 수학` 같은 **담당 과목 문구를 어디에도 표시하지 않음**(원장은 전체 관리자).
- 강사 계정은 그대로(`담당: 영어 중고등` 등 유지). **원장만 예외.**
- 홈 부제·헤더·기타 모든 곳 점검.

## 3. UI 통일감 — 다 따로 노는 것 정리 ⭐
가장 큰 불만. 같은 요소는 **한 규격**으로.
- **날짜 형식 한 가지:** `2026년 6월 14일 (일)`. 영어의 네이티브 날짜/시간칸(`2026.06.14`·`-- --:--`) → **수학식 컨트롤**(‹ 날짜 › + 오늘로)로 교체.
- **부제(문구) 통일:** 개발 메모식 문구 삭제(아래 5번). 화면마다 "무엇을 하는 곳"만 한 줄.
- **출결 용어 통일:** 영어=`등원`/수학=`출석` → **한 단어로**.
- **카운트 배지 통일:** `26명`·`9` 등 위치·색·모양 한 규격.
- **출결 요약 카드**(등원/지각/결석) 양쪽 동일 적용.
- **빈 상태 통일:** `(없다는 한 줄) + (행동 버튼)` 패턴. (예: `오늘 등원 예정 학생이 없어요. [시간표 보기]`)
- **메뉴명 통일:** 영어 `오늘 (일일기록)` → `오늘`(뉘앙스는 부제로). 사이드바 그룹명 `영어 수업관리 — 중고등` → `영어 수업관리 (중고등)`. 브레드크럼 구분자 `·`로 통일.
- *완료 기준:* 수학·영어·공통 화면이 같은 골격·같은 컨트롤·같은 문구 톤으로 보인다.

## 4. 시간표 변경 요청 — 알림·승인·자동 팝업 (작동 수정) ⭐
- **현재:** 새 요청 폼(학생·과목·날짜·시간·**보낼 선생님 드롭다운(담당/지정)**·사유)·받은/보낸 탭 **존재**.
- **문제(보고):** 보내도 **알림이 안 울리고**, 받는 사람이 모름. 겹칠 때 **자동 팝업이 안 뜸**.
- **수정:**
  1. **알림 동작:** '변경 요청 보내기' → 보낼 선생님의 **'받은 요청'에 즉시 도착** + **헤더에 알림 배지(🔔)** 표시. 로그인 시 새 요청이 보이게.
  2. **승인 흐름:** 받은 요청에서 **승인/거절** → 승인 시 해당 날짜 시간표 **임시 반영**, 보낸 사람에게 **결과 알림**.
  3. **자동 충돌 감지 팝업:** 시간표/오늘에서 임시 변경 시 **영어↔수학이 겹치면** "시간표가 겹칩니다" 팝업 자동 표시 → **[변경 요청하기]** 누르면 이 새 요청 폼이 **자동 채워진 채** 열림.
  4. 날짜·시간 입력을 **앱 공통 컨트롤**로(네이티브 제거, 3번과 통일).
- *완료 기준:* 보내면 상대에게 알림이 뜨고 → 받은 요청에서 승인하면 시간표에 반영되고 → 겹칠 때 자동 팝업이 뜬다.

## 5. 문구·이름 정비 (민감 포함) ⭐
| 위치 | 현재 | 수정 |
|---|---|---|
| 영어 오늘/출결 부제 | `학습일지 중심 · 인센티브 없음 · 앱에 자체 저장됩니다.` | **삭제** → `오늘 등원 학생의 목표·숙제·코멘트를 기록하세요.` 등 화면 목적만 |
| 인센티브 | 영어 등에 '인센티브 없음' 노출 | **'인센티브' 단어를 수학 외 어디에도 쓰지 않음**(없는 걸 굳이 적지 않음) — 민감 |
| 영어 오늘 빈상태 | `왼쪽에서 학생을 선택하세요`(왼쪽 빔) | `오늘 등원 예정 학생이 없어요. '추가 등원'으로 추가하세요.` |
| 업무보드 빈칸 | `비어 있음` | `아직 없어요` |
- 어드민인 만큼 **모두가 이해되는 쉬운 말**, 같은 동작=같은 단어.

## 6. 편의·퀵 (이미 있는 것 다듬기 + 추가)
- 전역 학생 검색·즐겨찾기(★) **동작 완성도** 점검(빠르게 학생/메뉴로 점프).
- 날짜 있는 화면 전부에 **'오늘로'** 버튼.
- 출결 **키보드 단축키**(1 출석·2 지각·3 결석·Enter 다음) — 빠른 입력.

## 7. 디자인 폴리시 (깔끔·멋지게)
- 글이 선/모서리에 붙지 않게 **여백**(카드 안쪽 16~20px, 8px 리듬).
- 상태색(출석 초록·지각 앰버·결석 빨강)·배지·카드 **한 규격**.
- 다크 토큰은 「다크모드 핸드오프」, 폴리시 세부는 「v3」 따름.

## 작업 순서
1. **원장 담당 표기 제거 + 홈 재정리**(2·1번) — 첫인상.
2. **문구·인센티브 정비**(5번) — 민감·빠름.
3. **UI 통일**(3번) — 날짜·배지·빈상태·메뉴명.
4. **시간표 변경 요청 알림·승인·자동 팝업**(4번).
5. **편의·퀵 + 디자인 폴리시**(6·7번).
6. 각 단계 **수학 회귀 확인**.
