# 핸드오프: 쏘이지(Soez) 디자인 시스템

> **바꿈영수학원 수업 관리 앱** — 브랜드 & 디자인 시스템 구현 지시서
> 컨셉: "부지런한 벌처럼, 꿀처럼 달콤한 보상. 쓰기 정말 쉬운(so easy) 수업 관리."

---

## Overview

이 문서는 학원 수업 관리 앱 "쏘이지"의 브랜드·디자인 시스템을 코드로 구현하기 위한 지시서입니다. 꿀벌(부지런한 학생)과 꿀(보상 포인트)을 하나의 컨셉으로, 로고·색·타이포·마스코트부터 UI 컴포넌트·화면 구석 디테일까지 한 규칙으로 정리했습니다.

## About the Design Files

이 번들의 파일들은 **HTML로 만든 디자인 레퍼런스**입니다 — 의도한 모양과 동작을 보여주는 프로토타입이지, 그대로 복사해 배포할 프로덕션 코드가 아닙니다. 할 일은 **이 디자인을 타깃 코드베이스의 기존 환경(React, Vue, SwiftUI, 네이티브 등)에서 그 환경의 패턴·라이브러리로 재현**하는 것입니다. 아직 환경이 없다면 프로젝트에 가장 적합한 프레임워크를 골라 구현하세요.

토큰 값(색·타이포·간격·radius)은 정확한 수치이므로 그대로 사용하면 됩니다.

## Fidelity

**High-fidelity (hifi)** — 최종 색·타이포·간격·인터랙션이 확정된 픽셀 단위 목업입니다. 코드베이스의 기존 라이브러리로 UI를 픽셀에 가깝게 재현하세요.

## 1. 브랜드 컨셉

| 의미 | 내용 |
|---|---|
| So Easy | "쓰기 정말 쉽다" — 앱의 핵심 가치 |
| 이지(현) | 제작자 이지현 원장의 이름 |
| 쏘이지 🐝 | "벌에 쏘이지" — 꿀벌 연상에서 컨셉 탄생 |

- **부지런한 학생 = 벌**, **보상 포인트 = 꿀**, **반·명단 = 벌집**. 이 은유를 UI 전반에 일관되게 적용합니다.

## 2. 로고

같은 벌 심볼을 기준으로 자리 크기에 따라 세 형태를 사용합니다.

- **메인 로고 (큰 자리):** 벌 심볼 + "쏘이지" 워드마크(가로 배치) + 보조 텍스트 "SOEZ · so easy"
- **영문 변형 (굿즈):** 큰 대문자 **S** + 벌 심볼(= "o" 자리) + "ez" → "S🐝ez" 로 읽힘
- **앱 아이콘 (작은 자리):** 꿀색 그라데이션 라운드 사각형(radius 26px, border 1.5px #2A2926) 위 흰색 벌 심볼

### 폰트 규칙 (중요)
- **로고 워드마크("쏘이지", "Soez")만 Jua(주아)** 사용
- **그 외 모든 텍스트(UI·본문·제목)는 Pretendard** — 가독성 우선

### 로고 사용 규칙
- ✅ 여백은 벌 몸통 높이의 1배 이상 확보
- ❌ 벌 줄무늬 색·기울기 변형 금지
- ❌ 크림·꿀색 외 배경 위 직접 배치 금지(필요 시 반전형/아이콘 사용)

### 벌 심볼 구성
둥근 몸통(가로:세로 ≈ 16:14) + 차콜 가로 줄무늬 곡선 3선 + 작은 흰 날개 2장 + 더듬이 2개. 선 두께 위계: 외곽선 1.7 · 줄무늬 2.6 · 얼굴선 1.9. 작은 자리에선 날개·더듬이 생략 가능. (정확한 SVG path는 `Soez Design System.dc.html` 02·05 섹션 참고.)

## 5. 마스코트 "쏘이"

통통이의 둥근 비례 + 방긋이의 표정을 합친 통합 마스코트. **둥근 몸통·또렷한 줄무늬는 고정, 표정만 상황에 맞게 교체**합니다.

### 표정 세트 (6종)
| 표정 | 사용처 |
|---|---|
| 기본 | 기본 상태·중립 안내 |
| 방긋 | 환영·일반 긍정 |
| 윙크 | 가벼운 칭찬·팁 |
| 신남(응원) | 출석 완료·연속 출석 응원 |
| 생각 | 로딩·빈 입력 안내 |
| 축하 | 목표 달성·랭킹 1등 |

### 자리별 변형 (4종)
| 변형 | 형태 | 사용처 |
|---|---|---|
| 동글이 | 원형 컴팩트(날개 축소) | 작은 아이콘·뱃지 |
| 라인이 | 라인(아웃라인)만 | 미니멀 화면·비활성 |
| 꿀단지 | 벌 + 꿀단지 | 포인트·보상 화면 |
| 부지런이 | 날아가는 포즈 + 점선 궤적 | 로딩·동적 강조 |

## 6. 꿀·꿀벌 디테일 적용 규칙 (중요)

귀여운 디테일은 **꿀 포인트 / 학생 프로필 아바타 / 벌집 게이지 구간에만** 허용합니다. 업무용 통계·대시보드 그래프 등 공적 영역에는 적용하지 않습니다(아래 7번 참고).

| 요소 | 무엇 | 어디에 쓰나 |
|---|---|---|
| **벌집 아바타** | 사진 없을 때 육각형 칸 + 이름 초성 또는 벌 실루엣 | 학생 프로필·명단 카드 |
| **꿀방울 포인트** | 포인트 단위 = "꿀", 숫자 옆 꿀방울 아이콘. 적립 시 `+15` (success 색) | 포인트 표시 어디서나 |
| **벌집 게이지** | 막대 대신 벌집 칸이 하나씩 차오르는 진행률 (예: 7/12칸) | ① 학생 프로필 내 목표 진행률 ② 출석 보상/포인트 화면. ❌ 대시보드 통계 그래프엔 사용 금지 |
| **여왕벌 랭킹** | 포인트 1등 아바타 위에 왕관 | 랭킹·리더보드 |
| **빈 화면** | 빈 벌집 + 작은 벌, 카피 "아직 모인 꿀이 없어요" | 데이터 없을 때 |

## 7. 공적 영역 가드레일 (업무용 화면)

- **월말 리포트:** 디자인 변경 금지 — 기존 그대로 유지.
- **대시보드 통계 그래프:** **형태(차트 종류·구조)는 유지**, 색상만 브랜드 톤으로 정리 가능. 벌집 게이지·마스코트 등 귀여운 장식은 넣지 않음. 학원 업무 도구로서 공적인 선을 지킬 것.

## 8. UI 컴포넌트 스펙

### 버튼
- **Primary:** bg `#F4A93C`, text `#2A2926`, border `1.5px #2A2926`, radius 12px, padding 11×20, weight 700, **입체 그림자 `box-shadow: 0 3px 0 #2A2926`**
- **Secondary:** bg `#fff`, text `#8A5A2B`, border `1.5px #C5851E`, radius 12px
- **Text:** text `#C5851E`, underline(offset 3px), no border
- **Sizes:** Small radius 10 / padding 7×14 / 13px · Large radius 14 / padding 14×26 / 17px·800
- **Disabled:** bg `#EDE7DA`, text `#A8A296`, border `1.5px #DDD6C6`

### 입력 필드
- 기본: border `1.5px #E6E2D6`, radius 12px, padding 11×14, 15px
- 포커스: border `2px #F4A93C`, bg `#FFFCF5`, `box-shadow: 0 0 0 4px rgba(244,169,60,0.18)`
- 오류: border `2px #CE5246`, bg `#FCF0EF`, 하단 헬퍼 텍스트 `#CE5246` 12px
- 라벨: 13px / 600 / `#2A2926`

### 상태 뱃지 (pill)
- radius 999px, padding 6×13, 13.5px / 700, 앞에 7px 점(상태색)
- 출석 `bg #E4F0E2 / text #3C7A47`, 지각 `bg #FBE7DA / text #C25E2E`, 결석 `bg #F9E1DE / text #B23E33`, 보충 `bg #E1ECF1 / text #3A6883`, 꿀 `bg #FDEFD3 / text #8A5A2B`

### 학생 카드
- bg `#fff`, border `1px #E6E2D6`, radius 18px, padding 18px, `box-shadow: 0 2px 10px rgba(138,90,43,0.05)`
- 구성: 벌집 아바타(48×54) + 이름(16/800) + 상태 뱃지 + "반 · 🍯 N 꿀"(13/`#8A867B`) + 우측 액션 버튼
