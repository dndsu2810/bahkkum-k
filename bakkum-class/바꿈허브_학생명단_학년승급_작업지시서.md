# 학생 명단 — 학년 구조·필터·일괄 승급 작업지시서

**대상:** 학생 명단(공통)에 ①학년 2단 구조 ②세분화 필터 보기 ③학년 일괄 승급 추가.
> ⚠️ **수학 데이터 불변** 원칙 유지. 학년 필드만 바뀌는 거라 출결·숙제·진도 등 기존 데이터엔 영향 없게.

## 확정 사항 (지현님 결정)
- 학년: **직접 정하는 방식**으로 전환(생년월일 자동 → 수동 선택). *일괄 승급의 전제.*
- 일괄 승급 시 **고3 → '졸업'(또는 퇴원) 상태로 자동 전환**.
- 안전장치: **바로 적용 + 되돌리기 버튼**.

---

## 1. 학년 2단 구조 (구분 + 세부 학년) ⭐
- 현재: 구분(초등/중등/고등)만 저장, 학년("초6")은 **생년월일 자동 수식**.
- 변경: **세부 학년을 직접 선택**해 저장.
  - 초등: 1~6학년 · 중등: 1~3학년 · 고등: 1~3학년
  - 저장 형태: 구분 + 세부 학년(예: `초6`, `중2`, `고1`).
- **표시:** 명단·프로필·리포트 모두 `초6` 식으로 합쳐 표기.
- **기존 학생 초기값:** 현재 생년월일 기준 학년을 **1회 자동 채움** → 이후 수동 관리(자동 계산은 중단).
- 학생 수정(프로필)에서 구분 선택 시 **세부 학년 드롭다운**이 따라 나오게.
- *완료 기준:* 모든 학생이 "초등 + N학년"처럼 명확한 학년 값을 갖고, 수정에서 바꿀 수 있다.

## 2. 필터 보기 세분화 ⭐
학생 명단 상단에 필터 추가(기존 과목/구분 보기에 더해):
- **학교별:** 구룡초·판곡초·평동초·호평초·판곡중·호평중·평내중·판곡고·호평고 (노션 학교 옵션과 동일).
- **학년별:** 초1~고3 (2단 구조 기준).
- **겹쳐 보기:** 예) "호평초 + 6학년", "재원 + 중2"처럼 **여러 필터 동시 적용**.
- 필터 결과 위에 **인원수 표시**(예: "호평초 6학년 · 4명").
- *완료 기준:* 학교·학년(및 상태)로 걸러서 원하는 학생만 명단에 보인다.

## 3. 학년 일괄 승급 (반 승급) ⭐
- **버튼:** 학생 명단에 "**전체 학년 올리기**"(1년에 한 번 쓰는 진급용).
- **동작:** 모든 **재원** 학생 세부 학년 +1.
  - 초6 → **중1**, 중3 → **고1** 자동 연결.
  - **고3 → '졸업'(또는 퇴원) 상태로 자동 전환**(학년 더 안 올림).
  - 휴원/대기 학생 포함 여부는 선택 옵션으로(기본: 재원만).
- **안전장치:** 누르면 "○명 승급, 고3 ○명 졸업 처리됩니다" **확인 팝업** → **바로 적용** + 직후 **'되돌리기' 버튼**(직전 일괄 승급 1회 취소).
- (선택) 특정 학생만 빼고 승급: 승급 전 제외 체크.
- *완료 기준:* 버튼 한 번으로 전원 진급(고3 졸업 처리 포함)되고, 잘못 눌러도 되돌리기로 즉시 복구된다.

## 4. 학원 로고 업로드 (사이드바 "바" 자리) ⭐
- 현재 사이드바 좌상단의 **"바" 글자 박스** 자리에 **학원 로고 이미지**를 넣을 수 있게.
- **설정 화면**에서 로고 이미지를 **업로드/교체**(없으면 기본 "바" 박스 유지).
- 업로드하면 사이드바·헤더 등 로고가 쓰이는 곳에 반영. 권한: 원장.
- *완료 기준:* 설정에서 로고를 올리면 사이드바 "바" 자리가 그 로고로 바뀐다.

## 5. 학생 프로필 사진 (선택) ⭐
- 학생도 **프로필 사진**을 넣을 수 있게(학생 수정/프로필에서 업로드).
- **안 넣으면 기존 그대로**(이니셜 아바타). 강제 아님.
- 명단·프로필·리포트의 아바타 자리에 사진이 있으면 사진, 없으면 이니셜.
- *완료 기준:* 사진을 올린 학생은 사진이, 안 올린 학생은 기존 이니셜이 보인다.

---

## 작업 순서
1. **학년 2단 구조 전환**(1번) — 자동→수동, 기존값 초기 채움.
2. **필터 보기**(2번) — 학교별·학년별·겹쳐보기.
3. **일괄 승급 + 되돌리기**(3번).
4. **학원 로고 업로드**(4번) + **학생 프로필 사진**(5번).
5. 회귀 확인(수학 **데이터** 변화 없음, 학년 외 필드 영향 없음).

## 체크리스트
- [ ] 학년 = 구분 + 세부 학년(직접 선택), 표시 `초6` 식
- [ ] 기존 학생 학년 1회 자동 채움 후 수동 관리
- [ ] 학교별·학년별 필터 + 겹쳐 보기 + 인원수
- [ ] '전체 학년 올리기' → 확인 팝업 → 적용 → **되돌리기**
- [ ] 고3 → 졸업(또는 퇴원) 자동 전환
- [ ] 설정에서 학원 로고 업로드 → 사이드바 "바" 자리 교체(없으면 기본 유지)
- [ ] 학생 프로필 사진 업로드(선택) — 없으면 이니셜 그대로
- [ ] 수학 데이터·기타 필드 회귀 없음
