# 바꿈 허브 — UX 피드백 & 디자인 개선 작업지시서

**검토:** 정민아·원장 계정으로 전 화면(수학·영어 중고등·영어 초등) 둘러본 결과.
**목적:** 나중에 합쳐지며 생긴 **통일성 부족·불편·문구 문제**를 잡고, 깔끔하게 다듬는다.

> ⚠️ **절대 원칙 — 수학 불변:** 수학 화면·데이터·동작은 바꾸지 않는다. 통일은 "영어를 수학의 좋은 패턴에 맞추는" 방향으로.

---

## 0. 잘 된 점 (유지)
- 사이드바가 **수학 / 영어 중고등 / 영어 초등** 카테고리로 깔끔히 분리됨(접기 가능).
- 메뉴 세트가 과목별로 거의 통일됨(오늘·시간표·출결·숙제·진도·테스트·보강).
- 로고 아래 "정민아님, 담당: 수학 · 원장" 표시.
- 영어 출결의 **등원/지각/결석 요약 카드**는 한눈에 들어와 좋음.

## 1. ⚠️ 인센티브 — 수학 전용 (민감) ⭐
- **문제:** 영어 화면 부제에 "**인센티브 없음**"이라고 **글자로 적혀 있음.**
- **수정:** 인센티브는 **수학에서만** 존재. 영어·데스크·원장 등 **다른 어디에도 인센티브를 언급조차 하지 않는다.** "인센티브 없음"이라는 문구도 **삭제**(없는 걸 굳이 적지 않음).
- 원장 대시보드·집계 어디에도 영어 인센티브 항목이 끼지 않게.
- *완료 기준:* '인센티브'라는 단어가 수학 외 화면에 일절 등장하지 않는다.

## 2. 문구(마이크로카피) 정비 — "모두가 이해되게" ⭐
- **문제:** 부제가 개발 메모처럼 보임. 예: *"학습일지 중심 · 인센티브 없음 · 앱에 자체 저장됩니다."* → 사용자에겐 불필요/혼란.
- **수정:** 화면 부제는 **그 화면에서 뭘 하는지**만 사람 말로.
  - 영어 오늘 → "오늘 등원 학생의 목표·숙제·코멘트를 기록하세요"
  - 영어 출결 → "오늘 등원/지각/결석을 표시하세요. 결석은 보강 관리로 이어집니다."
- '앱에 자체 저장' 같은 **내부 동작 설명 삭제**(당연한 것).
- 학원 어드민인 만큼 **용어 통일**: 같은 동작은 같은 단어(기록/저장/완료 등).
- *완료 기준:* 모든 화면 부제·버튼·빈 상태 문구가 명확하고 일관됨.

## 3. 레이아웃 통일 — 수학 vs 영어 ⭐
관찰한 차이:
| 요소 | 수학(더 정돈됨) | 영어(덜 다듬어짐) |
|---|---|---|
| 날짜 이동 | `‹ 6월 14일 › + 오늘로` 화살표 | 네이티브 날짜칸(`2026.06.14`) — 투박 |
| 오늘 화면 | "오늘 등원 학생" 카드 + 빈 상태 CTA | "+추가 등원" 드롭다운 + "왼쪽에서 선택"(안내 약함) |
| 출결 | (요약 카드) | **등원/지각/결석 요약 카드(좋음)** |
- **방향:** 더 나은 쪽을 **양쪽 공통 패턴**으로 통일(내용은 과목별로 다르되 레이아웃은 동일).
  - 날짜 이동: **수학식 화살표 + 오늘로**를 영어에도.
  - 출결 요약 카드: **영어식 등원/지각/결석 카드**를 수학에도(통일).
  - 오늘 화면: 같은 카드 골격(헤더·등원 목록·빈 상태 CTA) 공유.
- *완료 기준:* 수학·영어가 같은 골격(헤더/날짜/카드/빈상태)으로 보이고, 내용만 과목별로 다르다.

## 4. 학생 명단 단일화 ⭐
- **문제:** 과목 그룹마다 '학생 관리'가 따로 있음(중복).
- **수정:** **공통 「학생 명단」 하나**로. (과목별로 나누지 않음 — 단일 출처 원칙)
- *완료 기준:* 학생 명단이 한 곳뿐이고, 과목 화면은 그 명단을 공유.

## 5. 빈 상태 개선
- 영어 오늘 "왼쪽에서 학생을 선택하세요" → 정작 왼쪽 목록이 비어 혼란.
- **수정:** 빈 상태엔 **무엇을 하면 되는지** + 바로 행동 버튼(예: "오늘 등원 학생 추가" / "시간표 보기"). 수학의 친절한 빈 상태와 통일.

## 6. 퀵·편의 기능 (있으면 편한 것)
- **학생 검색(전역):** 상단에 검색 → 학생 이름으로 바로 점프.
- **즐겨찾기:** 자주 쓰는 메뉴 별표(이미 별 아이콘 있음 → 동작 완성).
- **'오늘로' 버튼**을 날짜 있는 모든 화면에.
- **빠른 추가:** 어느 화면에서든 학생/숙제 빠르게 추가.
- **출결 키보드 단축키**(1 출석·2 지각·3 결석·Enter 다음).
- **최근 본 학생/화면** 빠른 링크.

## 7. 디자인 폴리시 (깔끔·멋지게)
- 카드 안쪽 여백 넉넉히(글이 선에 붙지 않게), 8px 간격 리듬.
- 날짜 컨트롤·요약 카드·배지 **한 규격**으로 통일.
- 상태색(출석 초록·지각 앰버·결석 빨강) 일관 적용.
- 세부는 「v3 통합 업그레이드」·「다크모드 핸드오프」 토큰을 따른다.

## 작업 순서
1. **인센티브 문구 제거**(1번) — 민감, 최우선.
2. **문구 정비 + 빈 상태**(2·5번).
3. **레이아웃 통일**(3번) — 날짜·요약카드·오늘 골격.
4. **학생 명단 단일화**(4번).
5. **퀵·편의 기능**(6번).
6. **디자인 폴리시**(7번).
7. 각 단계 **수학 회귀 확인**.

- [ ] 수학·영어 레이아웃 통일(날짜 화살표·출결 요약카드·오늘 골격)
