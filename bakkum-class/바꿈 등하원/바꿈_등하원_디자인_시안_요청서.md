# 바꿈 등하원 — 디자인 시안 요청서 (클로드 디자인용)

> 수업관리 앱에 들어갈 "등하원" 화면들의 디자인 시안을 요청합니다.

## 무엇을 디자인하나

바꿈 수업관리 앱에 새로 들어갈 "등하원" 화면 3종의 시안.

## 브랜드 가이드

- 학원: 바꿈영수학원
- 컬러: 네이비 + 슬레이트 블루 계열에 화이트
- 로고: 네이비 배경의 흰 고래
- 분위기: 깔끔하고 정돈된, 신뢰감 있는 느낌
- 이모지는 쓰지 말 것. 필요하면 아이콘으로 표현.
- 기존 수업관리 앱의 디자인 톤과 일관되게.

## 화면별 요청

### 1. 선생님용 등하원 관리 (가장 중요 · 메인)

- 매일 여러 번 빠르게 훑는 화면이라, 한눈에 스캔이 잘 되는 게 1순위.
- 상단: 오늘 날짜 + 등원/하원/미발송 요약 숫자.
- 학생 목록(카드 또는 행) 항목: 이름, 상태 배지(등원완료 / 하원완료 / 미발송 / 정정), 시간, "알림 보내기" 버튼.
- **미발송은 빨간색**으로 확 눈에 띄게.
- 데이터가 많아도 안 길어지게 날짜별 접기를 고려.
- 정보 밀도는 있되 답답하지 않게.

### 2. 학생용 찍기 (태블릿)

- 아이들이 쓰는 화면이라 크고 단순하게.
- 본인 찾기(반별 또는 검색) + 큼직한 "등원" / "하원" 버튼.
- 누른 뒤 "○○ 등원 완료" 같은 큰 확인 표시.
- 터치 친화적, 글씨 크게.

### 3. 학생 상세 — 등하원 이력 (조회용)

- 깔끔한 타임라인 또는 표.
- 학부모 상담 때 보여줄 용도라 정돈된 느낌.

## 디바이스

- 선생님 관리 화면: PC / 태블릿
- 학생 찍기 화면: 태블릿
- 학생 상세 이력: 기존 수업관리 화면과 같은 환경

## 참고 (구조 맥락)

- 같은 등하원 데이터를 "오늘 전체"(선생님 관리 화면) 각도와 "학생 한 명"(상세 화면) 각도로 다르게 보여주는 구조.
- 학생이 찍는 행위 → 선생님이 확인·발송 → 학부모 카톡 알림 으로 이어지는 흐름.
