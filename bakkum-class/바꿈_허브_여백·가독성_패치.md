## 패치 0 — 다크모드 대비 (⭐최우선, 전 화면 동시 효과)

**증상:** 라이트모드는 글자 대비가 잡혀 있으나(C-1), 다크모드는 본문·안내문이 배경에 묻히고 **카드 테두리가 배경과 거의 같은 색**이라 카드들이 한 덩어리로 보임.

**위치:** `src/styles.css` 의 `[data-theme="dark"]{ … }` 토큰 블록.

**조치:** 현재 값을 확인하고, 아래 **목표 대비**가 되도록 조정(값은 가이드, ±한 단계 허용):

```css
[data-theme="dark"]{
  --ink2:#c7d0dd;   /* 본문 — 한 단계 밝게 */
  --ink3:#9aa6b8;   /* 보조설명 최저선. 이보다 어두우면 "검사할 숙제 없음"류가 묻힘 */
  --line:#2c3a52;   /* 카드/구분선 경계 또렷하게 */
  --card:#161f30;   /* 카드를 페이지 배경(#10151f)보다 한 단계 밝게 → 입체감 */
}
```

**확인:** 다크에서 ① 빈/안내 문구가 읽히는지 ② 카드끼리 경계가 보이는지 ③ 라이트모드는 그대로인지. **이 한 패치만으로 체감 절반은 해결됨.**

## 패치 1 — "오늘 등원 학생" 입력 패널 (여백·구조)

**증상:** 우측 입력 영역에서 출결 선택 버튼(`출석/지각/결석`)이 **풀폭 빈 박스 안 왼쪽에 몰려** 있어 가로는 텅 비고, 섹션 세로 간격은 들쭉날쭉.

**위치 찾기:** "오늘 등원" 화면 컴포넌트(`src/Workspace.tsx` 또는 today 관련 섹션)와 `styles.css`의 `.today-*` 규칙. 출결 세그먼트는 기존 `.today-seg`/`.att-seg`(트랙형) 스타일이 이미 있으니 **재사용**.

**조치 (값만):**
- **1-a 콘텐츠 폭 제한** — 입력 패널(출결·숙제 영역)이 화면 끝까지 늘어나지 않게 `max-width:640px`.
- **1-b 출결 세그먼트** — 풀폭 빈 박스를 버리고 **내용 폭만큼만** 차지하는 트랙형으로. 버튼 `min-height:44px`, 버튼 사이 `gap:8px`, 미선택 버튼도 옅은 배경(`#1f2a3d` 수준)으로 "눌 수 있음"을 표시. (선택 시 색 채움은 기존 `.on-present/on-late/on-absent` 그대로)
- **1-c 섹션 리듬 통일** — `출결` / `검사할 숙제` / `내줄 숙제` **섹션 사이 `24px`**, **라벨→내용 `8px`** 로 고정.
- **1-d 섹션 분리** — 각 섹션을 `1px solid var(--line)` + `border-radius:var(--r-md)` 옅은 서브카드로 감싸 눈이 구분하게.
- **1-e 빈 상태** — "오늘 검사할 숙제 없음"을 점선 박스(`1px dashed var(--line)`) + 글자 `var(--ink3)`로.

**확인:** 가로 빈 공간이 사라지고 섹션이 또렷이 구분되는지. 입력 동작·저장 로직은 **그대로**.

## 패치 2 — 연간 수업 계획표 그리드

**증상:** 빈 칸마다 `+` 기호 수십 개가 노이즈 → 실제 일정이 묻힘. 행 높이는 크고 글자는 가로로 눌림. 셀 전체 옅은 파랑 채움이 텍스트를 묻음.

**위치 찾기:** 연간 계획표 그리드 컴포넌트와 그 셀/추가버튼 클래스(`styles.css`에서 해당 화면 규칙 검색).

**조치:**
- **2-a `+` 노이즈 제거** — 빈 셀의 `+` 버튼 `opacity:0`, **셀 hover/focus 시에만** 표시(`opacity:.45`). 클릭 영역(추가 기능)은 그대로 유지.
- **2-b 행 높이** — 내용에 맞춰 축소, 셀 내용 `vertical-align:middle` / 세로 가운데. 과한 세로 여백 제거.
- **2-c 카테고리 표현 변경** — 셀 전체 옅은 파랑 채움 대신, **행 머리 왼쪽에 `4px` 컬러 바**(카테고리 색)로. 채움이 빠지면 텍스트가 또렷.
- **2-d sticky 고정** — 첫 열(항목명)과 월 헤더 행을 `position:sticky`(left:0 / top:0)로 → 가로 스크롤해도 어느 행·월인지 유지.
- **2-e 이번 달 강조 컬럼(6월)** — **잘 돼 있으니 유지.**

**확인:** `+`가 사라져 일정만 보이는지, 한 행을 12개월 가로로 눈으로 따라가지는지.
